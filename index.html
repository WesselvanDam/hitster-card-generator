CSV Format Guide

This tool requires you to upload a CSV file containing the data that you want to turn into cards. While some websites allow you to link to a Spotify playlist, those may not get the song metadata correctly. Using a CSV, you have full control over the data and can customize it as needed.

Required Columns:

  • type: Category (e.g., song, painting). Cards within each category can get their own color on the 'front' of the card (the side without the QR code).
  • url: Valid URL for QR code. For example, a Spotify link or a link to an image (make sure the link is publicly accessible).
  • top, center, bottom: Text content fields. In the official Hitster game, these contain the song title, year, and artist respectively.

Example CSV:

type url top center bottom
song https://youtu.be/example1 Song Title 1985 Artist Name
painting https://example.com/art.jpg Starry Night 1889 Van Gogh
