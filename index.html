Card PDF Generator

Instructions

CSV File Format Requirements:

  • File must be in CSV format (comma or semicolon separated)
  • Required columns: type, url, top, center, bottom
  • type can be any text value (e.g., song, video, article, painting, etc.)
  • url must be a valid URL
  • top, center, and bottom are text fields for card content
  • center can be empty for cards without center text

How to use:

  1. Upload your CSV file
  2. Configure colors for each card type found in your CSV
  3. Adjust other configuration parameters if needed (optional)
  4. Click "Generate PDF" to create your cards
