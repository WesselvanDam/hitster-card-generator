CSV Format Guide

This tool requires you to upload a CSV file containing the data that you want to turn into cards. While some websites allow you to link to a Spotify playlist, those may not get the song metadata correctly. Using a CSV, you have full control over the data and can customize it as needed. Feel free to check out our example CSV file.

Required Columns:

  • type: Category (e.g., song, painting). Cards within each category can get their own color on the 'front' of the card (the side without the QR code).
  • url: Valid URL for QR code. For example, a Spotify link or a link to an image. Make sure the link is publicly accessible, and that the link text does not spoil any answers!
  • top, center, bottom: Text content fields. In the official Hitster game, these contain the song title, year, and artist respectively.

Example CSV:

type url top center bottom
song https://youtu.be/example1 Song Title 1985 Artist Name
painting https://example.com/art.jpg Starry Night 1889 Van Gogh

Advanced Settings

Configure bleed, margins, and card sizes.

Configuration Guidelines

  • Paper Width/Height: Standard A4 is 210×297mm.
  • Bleed: Extra space around the page edge for printing (typically 3mm). This is usually necessary if you're printing the PDF at a professional print shop. Set it to zero if you're printing at home.
  • Card Size: The size of each square card on the page. 66 mm is about the same size as the original Hitster cards.
  • Card Gap: Space between cards for cutting.
  • Card Margin: Space from the page edge to the first card.

Constraints

To be able to align both sides of the card for double-sided printing, there are specific constraints: The paper width minus twice the card margin must be exactly equal to N times the card size plus N minus 1 times the card gap.
With the default values, this means:
(210mm - 2×3mm) = 3 × 66mm + 2 × 3mm
You probably want to stick to the default values, but changing them might be necessary if you want differently sized cards or are using non-standard paper sizes.

Pro Printing Tip

For a retail feel, use thick cardstock (around 300g/m²). Consider using a paper cutter for perfectly clean edges and uniformity.
